1. Instalar modulos del kernel
sudo apt install linux-headers-$(uname -r)

2. Compilar el modulo
make

3. Cargar el modulo
sudo insmod ch341.ko
 
(para DESINSTALAR)
sudo rmmod ch341

4. Verificar que el modulo se cargo correctamente
lsmod | grep driver

(revisar los logs)
dmesg | tail

5. Crear el archivo de dispositivo
ls /dev/ttyUSB*

6. Cargarlo automaticamente
sudo cp driver.ko /lib/modules/$(uname -r)/kernel/drivers/usb/serial/
sudo depmod
